# Pequeño ejemplo de una API RESTful en Node.js usando Fastify, MongoDB y Swagger

## Comandos

### Instalar dependencias

    npm i

### Ejecutar en modo desarrollo (con nodemon)

    npm run dev

### Ejecutar

    npm start
